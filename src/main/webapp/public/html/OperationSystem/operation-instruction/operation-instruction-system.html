业务流程

账号说明

  • 账号：聚橙公司邮箱，eg，wanglingling@juooo.com；
  • 密码：默认密码为员工身份证后六位，eg，203627，登录后请修改密码；
  • 权限：权限分配需要联系运营部同事。

新增项目

  • 第一步： 在项目列表点击“新增”

  • 第二步：填写表单、勾选项目所需资料

  • 第三步：点击“保存”或者“保存并提交”

    注1：点击“保存”后，只有自己可见，存在草稿箱里。
    注2：点击“保存并提交”，则提交到审核员审核。审核通过后则大家可见，审核不通过则回退到草稿箱。

上排期/修改排期信息

  • 第一步：在项目列表，勾选一个项目，并点击“上排期”

  • 第二步：填写基本信息、装走拆台时间、所需资料配置

  • 第三步：“保存”或者“保存并提交”

    注1：点击“保存”后，只有自己可见，存在草稿箱里；
    注2：点击“保存并提交”，则提交到审核员审核。审核通过后则大家可见，审核不通过则回退到草稿箱。

  • 第四步：保存后，可修改排期信息

    当“演出场馆”和“演出时间”修改后，需要再次提交审核。

添加资料

  • 第一步：打开“资料管理”-“预算/合同/报批/宣传/票图/执行”，勾选一条内容，点击“编辑”

  • 第二步：上传资料，并保存

注：若该资料已经完成，请务必手动勾选“完成”，否则保存后，资料状态仍然是“未完成”。

录入销售数据

  • 第一步：选择需要录入的“排期”和“销售日期”，点击“录入”

  • 第二步：录入数据，并保存

注：仅录入赞助金额、置换金额，其他销售数据ERP录入后会自动推送。

查看/导出销售报表

  • 一：打开“区域销售/剧院销售/项目销售”，可以查看概况。若要查看详情，点击具体条目：

  • 二：导出排期数据，点击导出，将直接导出，所有搜索结果

  • 三：查看排期销售详情，可导出报表

新增/修改配置

  • 第一步：打开要修改的配置

  • 第二步：填写必填项，并保存

注1：已经被引用过的配置项不能删除。 注2：“已被引用”名称解释，举例而言，比如，国家配置项其一是“法国”，而供应商列表中至少有一个供应商的国籍选择了法国。则“法国”就算被引用了，便不能再删除。 注3：配置项修改需谨慎，举例而言，比如，国家配置项其一是“法国”，若修改为“法兰西”，则所有选择了国籍为法国的供应商，其国籍自动变为“法兰西”。

新增/修改角色权限

  • 第一步：点击“新增”

  • 第二步：填写基本信息、勾选允许访问的页面

注1：区分城市，举例而言，
角色A，勾选了区分城市；
账号A，它的角色是“角色A”，它的城市属性是上海；
那么账号A在上排期或者查看排期时，只能添加/查看城市是“上海”的排期；
注2：区分事业部/子公司，举例而言，
角色A，勾选了区分事业部/子公司；
账号A，它的角色是“角色A”，它的事业部/子公司是万有天橙；
那么账号A在上排期或者查看排期时，只能添加/查看事业部/子公司是“万有天橙”的排期；

模块：项目管理、排期管理、资料管理……
栏目：预算、合同、报批……
